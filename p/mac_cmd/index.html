常用指令

command 用途
clear clear terminal window
ls list directory content
ls -a 將隱藏檔案也列出
cd change directory to home
cd .. change directory to parent
cd - change directory to last working directory
cd / change directory to root
pwd print working directory
rm file_name remove file
mkdir dir_name make directory
rmdir dir_name remove directory
rm -R dir_name remove the directory and all files in the directory
cp -R A B 將 A 資料夾的東西全部複製到 B
mv A dir_name 將 A 搬到資料夾
cat file_name 查看內容
grep text file_name 抓取內容
