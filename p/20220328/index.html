高等 UNIX 筆記

setjmp and longjmp function

  • goto，只能在同個 function 裡使用。
    • 在系統程式很常使用，例如錯誤處理。
  • 跨 function 時，用 setjmp 和 longjmp。

用法

  • int setjmp(jmp_buf env);
    • 第一次 call 時，return 0
  • void longjmp(jmp_buf env, int val);
    • val will be returned from setjmp

Stack Frame

  • args, ret, local var

Restoration

  • 變數放在 register 會被還原，但放在 memory 不會。
  • register int a;
  • violatile int b;

Process Creation

fork

  • pid_t fork(void)
  • returns: 0 child, pid of child in parent, -1 on error
  • exec
  • vfork, clone

Process Termination

  • SIGCHLD
  • wait, waitpid
  • zombie process
    • avoid zombies by calling fork twice

Race Condition

  • 不知道 child, parent 哪個會先跑
  • 解法：
    • 等 child 全部結束才讓 parent 跑
    • inprocess communication

Process Execution

  • exec
