在 Mac 安裝 Docker

  • 下載 Docker Desktop
  • 創一個資料夾，下載 Dockerfile 和 docker-compose.yml
  • docker-compose up -d
  • 用 ssh kylu@localhost -p20022 連線
  • 在 /.data/home 下創建使用者，讓資料可以跟本機相通
  • 用 passwd 設定密碼
