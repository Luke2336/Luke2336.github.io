自我介紹

Name 姓名

  • Wie heißen Sie / heißt du?
  • Wer sind Sie / bist du?
  • Ich bin/heiße ___.
  • Mein Name ist ___.

Herkunft 籍貫

  • Woher kommen Sie / kommst du?
  • Ich komme aus Taiwan.

Wohnort 居住地

  • Wo wohnen Sie / wohnst du?
  • Ich wohne in ___.

Alter 年齡

  • Wie alt sind Sie / bist du?
  • Ich bin 20 Jahre alt

Beruf 職業

  • Was sind Sie / bist du von Beruf?
  • Was machen Sie / machst du beruflich?
  • Ich bin / arbeite als ___ bei ___.

Famienstand

  • Ich habe ein Kind / zwei, keine Kinder.

  • Das ist ein Sonderangebot（特價）.
  • Das ist (sehr / zu / aber) teuer / günstig / billig.
  • Der Tisch ist zu groß / zu klein.
  • Ich finde die Lampe (wirklich) sehr schön.
