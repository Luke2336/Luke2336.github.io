dynamic, final, const

  • 沒有被宣告類型的變數會被宣告為 dynamic。
  • final 和 const 都不可以修改，const 用於編譯時期常數。
  • final(const) dataType name
1void main() {
2    dynamic x = 10;
3    x = 20;
4    final y = x;
5    const z = 10;
6}
