OS HW4 - Meltdown

  • 要在能特定的 CPU 上才能 Meltdown，大部分 Intel 的 CPU 都可以。
  • Meltdown
    • 現代的 CPU 多透過 Pipeline 去執行指令，當其中某些較快的指令執行完畢後，就會發出 Out-of-order Exception。再 CPU 處理 Exception 前，資料會先從 Register 複製到 Cache。
    • Meltdown 透過某些指令將 Cache 裡的東西搬到 User Mode 能存取的記憶體裡。
