電路學 CH13

Self Inductance

$$L = N \frac{d \phi}{di}$$

Mutual Inductance

$$M = M_{12} = M_{21}$$

$$v_2 = N_2 \frac{d \phi_{12}}{dt} = M_{21}\frac{di_1}{dt}, \ M_{21} = N_2 \frac{d \phi_{12}}{di_1}$$

Coupled Circuit

$$v_1 = L_1 \frac{di_1}{dt} + M \frac{di_2}{dt}$$

$$\mathbb{V}_1 = j \omega L_1 \mathbb{I}_1 + j \omega M \mathbb{I}_2$$

Energy

$$w = \frac{1}{2} L_1 I_1^2 + \frac{1}{2} L_2 I_2^2 \pm M I_1 I_2$$

Coupling Coefficient

$$k = \frac{M}{\sqrt{L_1 L_2}} = \frac{\phi_{12}}{\phi_{11} + \phi_{12}} = \frac{\phi_{21}}{\phi_{21} + \phi_{22}} \in [0, 1]$$

Perfect Coupling

$$k = 1 \iff \phi_{11} = \phi_{22} = 0$$

Linear Transformers

$$\begin{aligned} \mathbb{Z}_{in} &= \mathbb{Z}_P + \mathbb{Z}_R \\ &= (R_1 + j \omega L_1) + \frac{\omega^2 M^2}{R_2 + j \omega L_2 + \mathbb{Z}_L}\\ \mathbb{Z}_P &:\text{primary impedance}\\ \mathbb{Z}_R &:\text{reflected impedance} \end{aligned}$$

Ideal Transformers

  • $$\begin{cases} L_1, L_2, M \to \infty\\ k = 1 \iff M = \sqrt{L_1 L_2} \\ R_1 = R_2 = 0 \end{cases}$$
  • $$\mathbb{V}_2 = \sqrt{\frac{L_2}{L_1}} \mathbb{V}_1 = n \mathbb{V}_1$$
  • Turn Ratio $$n = \frac{\mathbb{V}_2}{\mathbb{V}_1} = \frac{\mathbb{I}_1}{\mathbb{I}_2} = \frac{N_2}{N_1} = \sqrt{\frac{\mathbb{Z}_L}{\mathbb{Z}_{in}}}$$
  • Make Ideal !! $$\omega L_2 \gg |\mathbb{Z}_L|$$
