設定 zsh

Jan 29, 2022
  • 下午將 shell 從 bash 改成 zsh。
  • 作業系統看到第 18 集。

設定 zsh

把 shell 換成 zsh

chsh -s /bin/zsh

安裝 Oh-My-Zsh

sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Terminal 顏色

  • 改成 sublime 的底色，字型改成白色。

設定 ~/.zshrc

  • 為了讓之前的 .bashrc 可以運作，於是要加上 source ~/.bashrc。
  • 主題選用 gozilla。
  • 選用了一些有趣的 plugins，例如 colored-man-pages、sublime、sudo、web-search、copydir、copyfile。
  • 改完後記得跑 source ~/.zshrc。
source ~/.bashrc
source ~/.bash_profile
