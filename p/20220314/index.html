高等 UNIX 筆記

  • sudo su -

File Descripter

  • 0: STDIN, 1: STDOUT, 2: STDERR
  • unistd.h
  • 0 ~ OPEN_MAX - 1，可用 setrlimit() 改（需要 root 權限）。
  • redirection, pipe

File I/O vs. Standard I/O

File I/O

  • via file descripter

Standard I/O

  • via wrapped file descripter
  • fileno()：給 FILE 指標，得到 descripter number

File I/O Function

open()

  • int open(const char *pathname, int flags, mode_t mode);
  • return descripter, error: return -1
  • flags
    • O_WRONLY | O_APPEND

creat()

close()

lseek()

  • off_t lseek(int fd, off_t offset, int whence);
  • offset 有正負
  • 32-bit
  • File Hole: 中間會填 0，磁碟機實際上只會用寫入的大小，檔案大小會包含空洞。
    • ls -s
    • cp file-hole file-nohole，沒辦法生出有洞的檔案。

read()

write()

File I/O Other Issues

Efficiency

  • buffer size 最好是 block size 的倍數。

File Sharing

  • Kernel Data Structure
  • fork
    • 如果兩個 process 都對同個檔案做操作，有可能會出問題。
      • Atomic Operation
    • pread, pwrite
  • dup, dup2
  • sync, fsync, fdatasync
  • fcntl: 對 descripter 下指令
    • O_NONBLOCK
    • val |= flag, val &= ~flag
  • ioctl: 對 device 下指令
  • dmesg
  • /dev/fd: virtual file system
    • cat /etc/passwd | cat /dev/fd/0
    • netstat

Standard I/O

Buffering

  • fully buffered: 減少 system call 數量
  • line buffered: 遇到換行做 system call
  • unbuffered
  • setbuf(), setbbuf()

Open Files

  • fdopen()

Tmp Files

  • tmpfile, mkstemp

Homework

  • uid 可從 /etc/passwd 取得
