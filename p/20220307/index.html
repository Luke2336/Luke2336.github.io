筆記 - 高等 UNIX

Filename

  • 不可以用 / 和 \0
  • 最多 255 characters

Directory

  • pwd
  • home path 存在 /etc/password

stat, fstat, lstat

  • file
    • data
    • metadata: file information（屬於目錄的 data）
  • stat：將指定的路徑的 metadata 撈出來存到 buffer
  • fstat：給開好的檔案
  • lstat：does not follow a symbolic link
  • OK: return 0, error: return -1

st_mode

File Type
  • Regular file
  • Directory file
  • Block special file
  • Character special file
  • FIFO
  • Socket
  • Symbolic link
 1#include "apue.h"
 2
 3int main(int argc, char *argv[]) {
 4	int			i;
 5	struct stat	buf;
 6	char		*ptr;
 7
 8	for (i = 1; i < argc; i++) {
 9		printf("%s: ", argv[i]);
10		if (lstat(argv[i], &buf) < 0) {   // -1
11			err_ret("lstat error");
12			continue;
13		}
14		if (S_ISREG(buf.st_mode))
15			ptr = "regular";
16		else if (S_ISDIR(buf.st_mode))
17			ptr = "directory";
18		else if (S_ISCHR(buf.st_mode))
19			ptr = "character special";
20		else if (S_ISBLK(buf.st_mode))
21			ptr = "block special";
22		else if (S_ISFIFO(buf.st_mode))
23			ptr = "fifo";
24		else if (S_ISLNK(buf.st_mode))
25			ptr = "symbolic link";
26		else if (S_ISSOCK(buf.st_mode))
27			ptr = "socket";
28		else
29			ptr = "** unknown mode **";
30		printf("%s\n", ptr);
31	}
32	exit(0);
33}
Permission
  • ls -la: owner, owner-group, other
  • UID, GID, EUID, EGID, SUID, SGID
  • r: 可讀, w: 可寫, x: 可執行
    • s: 可執行且開了 SUID 的功能
  • process 對檔案做操作是看 EUID, EGID
  • setuid, setgid, getgid, getegid
  • 12bit: suid, sgid, sticky, …
  • 執行檔跟目錄一定要有執行權限
  • 要刪一個檔案，只需要擁有目錄寫的權限
  • umask
  • chmod, chown
Sticky Bit
  • 對執行檔來說，用來決定執行完後，是否 Cache。
  • 對 directory 來說，只有 superuser, file owner, directory owner 可以 delete / rename 檔案。

File System

  • disk drivers, partitions, file system
  • i-node 放 metadata
  • 有些 i-nodes 有特殊用途
    • 0: 保留或不存在
    • 1: list of defective blocks
    • 2: root directory of a partition
  • i-node 不存檔名
  • directory blocks, data blocks
  • i-node 用完後，就沒辦法建檔案。
  • Reference count
    • 有多少個東西指向這個 i-node
    • hard link ln (RefCnt++)（不能跨 partition）
    • unlink rm (RefCnt--)
  • ls -li: 看 i-node 編號和 RefCnt

Symbolic link

  • ln -s
  • 不用管檔案在不在～

Directory Operations

  • mkdir, rmdir
  • opendir, readdir, closedir
  • rewinddir, telldir, seekdir
  • chdir
