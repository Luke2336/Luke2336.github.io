User Passwords

  • Old: salt + encrypted password（DES, 2 + 11 bytes）
  • New: Algorithm id + slat + encrypted password（雜湊值）（用 $ 分隔）
  • Common Algorithms: DES, MD5, SHA256, SHA512
  • hash(salt+pw)
  • crypt(const char *key, const char *salt)
    • crypt("password", "$1$abcdefg$")
    • -l crypt: 編譯時要 link lib
  • /etc/nsswitch.conf

Process Termination

  • exit 會從 start() 結束。

  • _exit 不會

  • Normal process termination in five Ways:

    • Return from main()
    • Calling exit
    • Calling _exit or _Exit
    • Return of the last thread from its start routine
    • Calling pthread_exit from the last thread

  • Abnormal process terminaion in three ways

    • Calling abort
    • Receipt of a signal
    • Response of the last thread to a cancellation request

atexit(void (*function)(void))

  • 先註冊的會後執行 (stack)

 2
 3void cleanup() {
 4    if (p != NULL) {
 5        p = NULL;

 7}

exit()

  • Call atexit registered functions
  • Performed a clean shutdown of the standard I/O library
  • fclose() all streams, remove tmpfile()
  • 都做完事後，會呼叫 _exit

_exit(), _Exit()

  • Terminate immediately

Environment Variables

  • env
    • 變數名稱＝變數的值
    • 用 $ 取得變數的值。
    • getenv()（每個平台都有）, putenv(), setenv(), unsetenv(), clearenv()。

Environment List

  • 用 C 字串紀錄。
    1. int main(int argc, char *argv[], char *envp[]);
    2. extern char **environ;
  • 記憶體不是唯讀。

Environment Functions

  • int unsetenv(const char *name);
  • int clearenv(void);

Memory Layout of a Program

  • Text segment (low addr.)
    • Code
  • Initialized data segment
    • static
  • Uninitialized data segment(bss)
    • 未賦值的陣列
  • Heap
    • Dynamic allocated memory
    • mmap：真正配置動態記憶體（限制小）
    • sbrk：改 upperbound，控制 heap 大小
  • Stack (high addr.)
    • Local variables, function call states
